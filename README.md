# BookBarcode
本コレクション  
読み終わった本・気になる本をバーコードで一括管理  

## アプリケーション概要  

バーコードをかざして読み終わった本、気になる本を管理するアプリ  

## 背景

自分の持っている書籍をデジタルで管理したい。  
気になる本をアプリ内で管理しておきたい。  
といった課題を解決するためのアプリ。  

## ポイント

本のバーコードを読み取ることで書籍情報を登録できる。  
登録した書籍は一覧として参照できる。
